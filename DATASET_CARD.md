# DATASET_CARD (DeFi-30)

## Dataset purpose
Benchmark detection quality for DeFi-specific smart contract vulnerabilities.

## Composition
- 30 contracts total
- Tier 1: Synthetic isolated patterns
- Tier 2: Real exploit recreations
- Tier 3: Public audit-derived real-world contracts

## Labels
- Canonical vulnerability class
- Severity
- Location
- Description

## Intended use
- Comparative evaluation of security tools
- Reproducible scoring and leaderboard submissions

## Out of scope
- Production exploit development
- Offensive automation
